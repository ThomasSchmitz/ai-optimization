# AI GEO - Generative Engine Optimization

An educational website designed to help marketing professionals optimize their product sales websites, social media presence, and online content for AI visibility in tools like ChatGPT, Gemini, Bard, Copilot, Grok, and AI-powered search overviews.

## About

AI Generative Engine Optimization (AI GEO) is the practice of optimizing your online presence to be more visible and accurately represented in AI-powered tools and search engines. As AI becomes the primary interface for information discovery, traditional SEO strategies must evolve.

This website provides comprehensive guides, best practices, and resources for marketing professionals to:

- **Optimize websites** for AI-powered search and discovery
- **Enhance social media presence** to be more visible to AI tools
- **Create content** that AI models can effectively cite and reference
- **Build authority** that AI systems recognize and trust
- **Monitor and improve** AI visibility over time

## Getting Started

Simply open `index.html` in your web browser to view the website locally.

```bash
# Clone the repository
git clone https://github.com/ThomasSchmitz/ai-optimization.git

# Navigate to the directory
cd ai-optimization

# Open in browser
open index.html
```

Or use a local server for development:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js http-server
npx http-server

# Then visit http://localhost:8000
```
